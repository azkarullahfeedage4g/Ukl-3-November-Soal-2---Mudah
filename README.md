# Ukl-3-November-Soal-2---Mudah
Program di Soal Ukl Nomor 2 ini bertujuan untuk mengetahui apakah sebuah bilangan dinyatakan ganjil atau genap
Cara kerja program ini adalah : 
Program ini menggunakan scanner sehingga pengguna dapat menginput sendiri bilangan yang ingin ditentukan ganjil atau genapnya
Setiap bilangan yang diinput akan dihitung dengan operator modulus (%) untuk mengambil sisa hasil pembagian bilangan 2
Jika sisa pembagiannya 0 maka bilangan tersebut adalah genap 
Jika sisa pembagiannya bukan 0 maka bilangan tersbut adalah ganjil
